* なにこれ
  みっくストアのプラグイン一覧に、私（katsyoshi）のプラグインを追加するプラグインです。

* インストール方法
  以下のコマンドを実行すればインストールできます。

  : $ mkdir ~/.mikutter/plugin && git clone git://github.com/katsyoshi/katsyoshi_store.git ~/.mikutter/plugin/katsyoshi_store
